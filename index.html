// CORE FEATURES

필요한 모든 기능이
하나에

복잡한 미팅 관리를 단순하게. 팀원 모두가 같은 정보를 실시간으로 공유합니다.

🏢
멀티 회사 전환

여러 사업체를 운영하거나 복수의 팀에 속해 있나요? 하나의 계정으로 여러 회사를 등록하고 원클릭으로 즉시 전환합니다. 회사별 데이터는 완전히 분리되어 보안이 보장됩니다. 초대 코드로 팀원을 추가하고 부서별로 색상을 지정해 관리하세요.

MULTI-TENANT
🎯
참석자 맞춤 알람

미팅에 등록된 참석자에게만 알람이 발송됩니다. 관련 없는 팀원에게는 알림이 가지 않아 불필요한 방해를 없앱니다.

TARGETED ALERT
⏰
3단계 스마트 알람

60분·30분·10분 전, 세 번에 걸쳐 알람을 보냅니다. 브라우저 알림과 Slack 메시지를 동시에 발송합니다.

AUTO ALERT
🔄
주간·반복 미팅 등록

매주 반복되는 스탠드업, 주간 보고를 한 번 등록하면 자동 생성됩니다. 요일 선택, 개별 날짜 수정도 자유롭게 가능합니다.

RECURRING
🔗
Zoom 링크 공유

미팅마다 Zoom 링크를 저장해두면 참석자 전원이 클릭 한 번으로 바로 입장할 수 있습니다. 링크를 찾아 헤매는 시간이 사라집니다.

ZOOM LINK
📅
3단계 일정 관리

개인 일정, 팀 공개 일정, 전사 공개 일정을 분리 등록합니다. 통합 캘린더에서 전체를 한눈에 확인할 수 있습니다.

3-LEVEL CALENDAR
💬
일정 댓글 기능

각 일정에 팀원들이 댓글로 소통할 수 있습니다. 장소 변경, 준비 사항, 공지 등을 일정 안에서 바로 공유하세요.

COMMENTS
✅
오늘 할 일 체크

오늘의 미팅과 일정을 한 화면에서 확인하고, 완료 체크로 진행 상황을 추적합니다. 진행중·예정·완료 상태를 실시간으로 파악합니다.

TODAY VIEW
📣
Slack 자동 알림

Webhook 연동으로 미팅 알람을 Slack 채널에 자동 발송합니다. 팀 전체가 미팅 전에 미리 인지할 수 있습니다.

// HOW IT WORKS

3분이면 시작합니다

설치 없이 웹 브라우저에서 바로 시작. 팀원 초대까지 5분이면 충분합니다.

01
회사 등록

회사명을 입력하고 계정을 만드세요. 자동으로 초대 코드가 발급됩니다.

02
팀원 초대

초대 코드를 공유해 팀원을 추가하세요. 부서별로 색상을 지정할 수 있습니다.

03
미팅 등록

날짜, 참석자, Zoom 링크를 입력하면 자동으로 알람이 설정됩니다.

04
자동 알람 수신

참석자에게만 미팅 전 단계별 알람이 발송됩니다. Slack으로도 동시에 알립니다.

지금 바로 시작하세요

무료로 사용 가능합니다. 별도 설치 없이 웹 브라우저에서 바로 실행되며, 모바일과 데스크탑 모두 지원합니다.
